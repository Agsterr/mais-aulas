# Mais Aulas — HTML

Exercícios práticos de **HTML** do curso da Alura — páginas estáticas para aprender estrutura, tags e semântica.

## O que contém

Páginas HTML com exercícios progressivos sobre:

- Estrutura básica de documentos HTML
- Títulos, parágrafos e listas
- Links e imagens
- Formulários
- Tabelas e organização de conteúdo

## Tecnologias

- HTML5
- CSS (básico, quando aplicável)

## Como visualizar

```bash
git clone https://github.com/Agsterr/mais-aulas.git
cd mais-aulas
```

Abra os arquivos `.html` diretamente no navegador ou use uma extensão Live Server no VS Code.
